# Интеграция с платёжными системами

## Статус
⚠️ **Планируется** - интеграция ещё не реализована.

## Назначение
Модуль предназначен для интеграции с платёжными системами (ЮKassa, Сбербанк, и т.д.) для обработки онлайн-платежей.

## Планируемая функциональность
- Создание платёжных ссылок
- Обработка webhook уведомлений о статусе платежа
- Возвраты и отмены платежей
- История транзакций

## Примечание
Модуль не подключён в `INSTALLED_APPS` и будет активирован при необходимости.
