# Интеграция с сервисами доставки

## Статус
⚠️ **Планируется** - интеграция ещё не реализована.

## Назначение
Модуль предназначен для интеграции с сервисами доставки (СДЭК, Почта России, курьерские службы) для автоматизации логистики.

## Планируемая функциональность
- Расчёт стоимости доставки
- Создание заявок на доставку
- Отслеживание статуса доставки
- Генерация документов (накладные, этикетки)

## Примечание
Модуль не подключён в `INSTALLED_APPS` и будет активирован при необходимости.
