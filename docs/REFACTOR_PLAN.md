## 5) Локальные проверки (делать по мере надобности и/или в конце крупного шага)

Backend:
- `python backend/manage.py check`
- `python backend/manage.py migrate`
- `python -m compileall backend`

Docker:
- `docker compose -f infra/docker/docker-compose.yml config`
- `docker compose -f infra/docker/docker-compose.yml up -d --build <service...>`
- `docker compose -f infra/docker/docker-compose.yml logs --tail=200 <service...>`

Health:
- `curl -i http://127.0.0.1:8000/healthz/`
