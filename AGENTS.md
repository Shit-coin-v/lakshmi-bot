## Обязательный порядок работы агента (STRICT)

Перед выполнением любых действий в репозитории агент ОБЯЗАН:

1. Прочитать файл `docs/AGENT_WORKLOG.md`, чтобы понять контекст последних действий.
2. Составить план текущей сессии в файле `docs/REFACTOR_PLAN.md`:
   - План должен содержать ровно 10 шагов.
   - План описывает ТОЛЬКО текущую сессию (не историю проекта).
   - Без внесения изменений в код.
3. Остановиться и дождаться подтверждения владельца проекта в чате.
4. Только после подтверждения — приступать к выполнению шагов плана.
